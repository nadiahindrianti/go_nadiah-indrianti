## Run Locally

Clone

```bash
  git clone https://github.com/nadiahindrianti/go_nadiah-indrianti
```

Status, Add, Commit

```bash
  git status
  git add . 
  git commit -m "add config file"
```

Push, Pull 

```bash
  git push origin main/name branch
  git pull
```

Branching

```bash
  git branch main/name branch
```
